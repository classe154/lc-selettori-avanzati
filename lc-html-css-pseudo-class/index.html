3. Pseudo-classe :hover

Regola: Si attiva quando il mouse è sopra l'elemento

a:hover - Passa il mouse sui link:

Link 1 - Hover me! Link 2 - Hover me! Link esterno
Nota: `:hover` funziona su tutti gli elementi, non solo i link!
Nota: Se un elemento ha `cursor: pointer`, non serve aggiungere altri stili :hover perché indica già che è interattivo.

div p:hover - Passa il mouse sui paragrafi dentro i div:

Paragrafo 1 - Hover me! (sono dentro un div)

Paragrafo 2 - Hover me! (sono dentro un div)

Paragrafo 3 - Hover me! (sono dentro un div)

Questo paragrafo è fuori dal div - non cambierà colore

4. Pseudo-classe :visited

Regola: Si applica ai link che sono già stati visitati

a:visited - Link visitati:

Link interno #1 Link interno #2 Google (probabilmente visitato) Sito mai visitato
Privacy note: I browser limitano gli stili :visited per motivi di privacy.
Spesso si può cambiare solo il colore, non altre proprietà come background o border.

5. Pseudo-classe :active

Regola: Si attiva quando l'elemento viene cliccato (mentre tieni premuto il mouse)

a:active - Clicca e tieni premuto sui link:

Clicca e tieni premuto! Anche questo! E questo!
Tip: `:active` dura solo mentre tieni premuto il mouse.
È molto veloce! Viene spesso usato per dare feedback visivo durante il click.

6. Pseudo-classe :focus

Regola: Si attiva quando l'elemento ha il focus (è selezionato/attivo per input)

input[type="text"]:focus - Clicca negli input per dare loro il focus:

Focus spiegato: Il "focus" significa che l'elemento è attivo e pronto a ricevere input. Per gli input questo succede quando ci clicchi dentro (vedrai il cursore lampeggiante). Il focus si può anche dare con il tasto Tab per navigare tra gli elementi.

Accessibilità: Il focus è importante per l'accessibilità! Le persone che navigano con la tastiera usano il Tab per spostarsi tra elementi focusabili.

7. Combinazioni di Pseudo-classi

Puoi combinare più pseudo-classi sullo stesso elemento!

  • Secondo elemento - Hover qui non farà nulla
  • Terzo elemento - Hover qui non farà nulla

9. Suggerimenti per il Testing

Come testare efficacemente:

  • :first-child/:last-child - Attiva il CSS e guarda quali elementi cambiano
  • :hover - Muovi il mouse sugli elementi
  • :focus - Clicca negli input o usa Tab per navigare
  • :active - Clicca e tieni premuto sui link
  • :visited - Visita alcuni link, poi ricarica la pagina

Pro tip: Usa gli strumenti per sviluppatori del browser (F12) per vedere le pseudo-classi in azione e per forzare stati come :hover e :focus per il testing!
