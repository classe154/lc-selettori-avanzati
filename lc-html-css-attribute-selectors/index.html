Template per Test Selettori di Attributi CSS

Aggiungi i selettori CSS nello stile per vedere gli effetti sui diversi attributi!

1. Selettore: tag[attr]

Regola: Seleziona elementi che hanno quell'attributo, indipendentemente dal valore

input[type] - Input che hanno attributo type:

2. Selettore: [attr="value"] - Valore Esatto

Regola: L'attributo deve avere esattamente quel valore

a[href="https://example.com"]:

Link esatto ✓
Con slash finale
Con pagina
HTTP invece di HTTPS

3. Selettore: [attr^="value"] - Inizia Con

Regola: L'attributo deve iniziare con quella stringa

p[class^="intro"]:

classe = "intro" ✓

classe = "intro-text" ✓

classe = "introduction" ✓

classe = "main intro" (non inizia con intro)

classe = "content"

4. Selettore: [attr$="value"] - Finisce Con

Regola: L'attributo deve finire con quella stringa

a[href$=".html"]:

Pagina HTML ✓
Pagina PHP
HTML con anchor (non finisce con .html)
HTML maiuscolo

5. Selettore: [attr*="value"] - Contiene Substring

Regola: L'attributo deve contenere quella stringa da qualche parte

p[class*="special"]:

classe = "special" ✓

classe = "very-special" ✓

classe = "special-item" ✓

classe = "my special class" ✓

classe = "normal"

6. Selettore: [attr~="value"] - Contiene Parola

Regola: L'attributo deve contenere quella parola completa separata da spazi

span[title~="importante"]:

title = "importante" ✓
title = "molto importante davvero" ✓
title = "importante per tutti" ✓
title = "informazioni-importante" (non separato)
title = "superimportante" (non separato)
